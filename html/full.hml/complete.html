1. Webpage

Definition

A webpage is a single document displayed in a web browser.

Uses

  • Display information
  • Create forms
  • Show media content

Example

Home Page, Contact Page, About Page

2. Website

Definition

A website is a collection of multiple webpages connected together.

Types of Websites

Type Definition Example
Static Website Content remains same for all users Portfolio Website
Dynamic Website Content changes based on user or database Instagram, Amazon

3. HTML

Definition

HTML stands for HyperText Markup Language. It is used to create webpages.

Uses

  • Create webpage structure
  • Add text, images, videos
  • Create forms and tables

Basic Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    Hello World
</body>
</html>

4. DOCTYPE HTML

Definition

The <!DOCTYPE html> declaration tells the browser that the document uses HTML5.

Uses

  • Enables HTML5 features
  • Helps browser render correctly

Example

<!DOCTYPE html>

5. Tags

Definition

Tags are keywords enclosed inside angle brackets.

Types of Tags

Type Example
Container Tag <p>Hello</p>
Empty Tag <br>

Common Tags

  • <h1> to <h6>
  • <p>
  • <br>
  • <hr>
  • <img>

6. Elements

Definition

An element includes opening tag, content and closing tag.

Example

<p>This is paragraph</p>

Difference Between Tags and Elements

Tags Elements
Only opening and closing keywords Complete structure with content

7. Block Level and Inline Elements

Block Level Elements

Block elements take full width.

Examples

<div>
<p>
<section>

Inline Elements

Inline elements take only needed width.

<span> <a> <b>

11. Attributes

Definition

Attributes provide extra information about HTML tags.

Global Attributes

Used in almost all tags.

Attribute Use
id Unique identification
class Grouping elements
title Tooltip text
style Inline CSS

Local Attributes

Used only in specific tags.

Tag Attribute
<img> src, alt
<a> href, target
<input> type, placeholder

15. Table

Definition

Table Tags

Tag Use

16. Media Tags

Definition

Media tags are used to add image, audio and video.

Tag Use
<img> Display image
<audio> Play audio
<video> Play video
<source> Multiple media sources

Image Example

Audio Example

<audio controls>
    <source src="song.mp3" type="audio/mp3">
</audio>

Video Example

<video controls width="400">
    <source src="movie.mp4" type="video/mp4">
</video>

17. Meta Tags

Definition

Meta tags provide information about webpage to browser and search engines.

Common Meta Tags

Meta Tag Use
charset Character encoding
viewport Responsive design
description SEO description
keywords Search keywords

Example

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
